桃園區處文件管理系統 - 憑證安裝包
=============================================

檔案說明：
  - TaipowerDMS-RootCA.cer  ← 根憑證（請由系統管理員放入此資料夾）
  - install-cert.bat        ← 一鍵安裝腳本（不需管理員權限）

使用方式（推薦：BAT 一鍵安裝）：
  1. 雙擊 install-cert.bat（不需以系統管理員身分執行）
  2. 看到「✓ 憑證安裝成功」訊息後即可
  3. 關閉所有瀏覽器後重新開啟，連線至 https://10.205.3.52:7443

備用方式（手動安裝）：
  1. 雙擊 TaipowerDMS-RootCA.cer
  2. 點擊「安裝憑證」
  3. 儲存位置選擇「目前使用者」→ 下一步
  4. 選「將所有憑證放入以下的存放區」→ 瀏覽
  5. 選「受信任的根憑證授權單位」→ 確定 → 下一步 → 完成
  6. 出現安全性警告時點「是」

注意事項：
  - 本安裝僅對「目前 Windows 使用者」生效
  - 若同一台電腦有多個 Windows 帳號使用本系統，每個帳號皆需安裝一次
  - 若您是系統管理員且希望全機生效，請聯絡 IT 部門透過 GPO 派發

如有問題，請聯絡系統管理員。
